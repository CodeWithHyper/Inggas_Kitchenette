# Ingga's Kitchenette Website
## Project Overview 
This website was developed by BSIT 3rd year students as part of their Web Development course. It showcases the catering packages, gallery of events, and client reviews of Ingga's Kitchenette, a dedicated catering service provider.
## Purpose 
To provide an online presence for Ingga's Kitchenette, highlight its specialized services (Catering, Special Events, Food Packs, Party Trays), and allow clients to request personalized price quotes directly through the website.

## Tools Used 
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL (via XAMPP/phpMyAdmin)
- Libraries: PHPMailer (for SMTP email), FontAwesome (for icons)
- Development: Visual Studio Code, Git & GitHub
## Site Structure 
- `index.html` – Main single-page application (Hero, About, Services, Gallery, Contact, Reviews)
- `db_connect.php` - Configuration file for database connection
- `assets/` – Folder containing images, logos, and icons
- `PHPMailer/` - External library for secure email handling
## Design Concept 
- **Color Scheme**: Deep Red (#d51500) and White to evoke appetite, passion, and warmth.
- **Typography**: Noto (Serif) for headings to add a touch of elegance, Lato (Sans-serif) for body text to ensure readability.
- **Layout**: Mobile-responsive design using Flexbox, featuring a sticky navbar and interactive cards.

- Full Stack Integration: Successfully connected an HTML frontend to a PHP/MySQL backend.
- Database Management: Learned to create tables and insert form data securely using Prepared Statements.
- Version Control: Learned how to deploy static sites via GitHub Pages 
- Responsive Design: Mastered CSS Media Queries to ensure the site looks good on mobile devices.
- Third-Party Libraries: Implemented PHPMailer to solve the limitations of standard PHP mail functions.
